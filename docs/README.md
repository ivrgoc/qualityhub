# QualityHub Documentation

This directory contains the documentation for the QualityHub project.

## Structure

```
docs/
├── README.md           # This file
├── api/                # API documentation
└── architecture/       # Architecture documentation
    └── adr/            # Architecture Decision Records
```

## Contents

### API Documentation

The `api/` directory contains API reference documentation. When running the backend locally, interactive API documentation is also available at:
- Swagger UI: http://localhost:3000/api/docs

### Architecture Documentation

The `architecture/` directory contains:
- System architecture diagrams and descriptions
- Architecture Decision Records (ADRs) in the `adr/` subdirectory
